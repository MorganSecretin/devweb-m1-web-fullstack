# LANCER LE PROJET
**Prérequis :**
- [Java 21](https://www.oracle.com/fr/java/technologies/downloads/#java21)
- [Maven](https://maven.apache.org/download.cgi)
- [Node.js](https://nodejs.org/fr/download)

**Commandes pour le lancement :**
```bash
# 1er terminal
cd ./backend-spring-boot
mvn spring-boot:run

# 2éme terminal
cd ./frontend-angular
npm start
```
